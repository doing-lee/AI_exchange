常见问题

  • AI ToolKit 能做什么？
    在本地连接交易所 API 后，可通过 MCP、CLI 或 Skills 方式，用自然语言完成行情查询、现货/合约下单、账户与策略管理等。支持 Claude、Cursor、终端等多种客户端。
  • 支持哪些 AI 客户端？
    MCP 方式支持 Claude、Cursor、Codex、OpenCode 等任意支持 MCP 的客户端；Skills 方式支持 OpenClaw 等支持 Skills 协议的客户端；CLI 则可在任意终端使用，无需 AI 客户端。
  • 这个工具是免费的吗？
    是的。AI ToolKit 完全开源，遵循 MIT 协议，可免费使用与二次开发。交易所 API 的费率以各平台规则为准。
  • 我的 API Key 安全吗？
    工具以本地进程方式运行在您的设备上，API Key 仅保存在本地，不会上传到第三方服务器。建议为 API Key 设置 IP 白名单、只读/交易权限等，并勿将 Key 泄露给他人。
  • 如果 AI 犯错，我会损失资金吗？
    AI 仅根据您的指令执行操作，下单前请仔细确认参数。建议先用小仓位或模拟环境验证，并为账户设置风控与限额，以降低误操作风险。
